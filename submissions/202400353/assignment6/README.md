# 🏎️ F1 경기 추천기 (F1 Race Recommender) 사용 가이드

**"어젯밤 F1 경기, 2시간 동안 다 볼 가치가 있을까?"**  
이 서비스는 머신러닝(Random Forest 회귀 모델)을 활용하여 F1 경기의 가상 평점를 예측하고, 바쁜 당신을 위해 **[전체 다시보기 vs 하이라이트]** 시청 가이드를 추천해 드립니다.

---

## 주요 기능

1.  **AI 평점 예측**: 경기 데이터(사고 횟수, 추월 수 등)를 기반으로 10점 만점의 재미 평점을 예측합니다.
2.  **과거 경기 데이터 불러오기**: 2021~2025년 실제 경기 데이터를 불러와, 당시의 재미를 AI가 어떻게 평가하는지 확인해 보세요.
3.  **가상 시나리오 시뮬레이션**: 
    *"만약 그 경기에서 사고가 한 번 더 났다면?"*
    <br> 라이더를 직접 움직여 가상의 상황을 만들고, 평점이 어떻게 변하는지 실험해 보세요.
5.  **맞춤형 시청 가이드**: 예측 점수에 따라 **[전체 관람 / 하이라이트 / 요약본]** 중 하나를 추천해 드립니다.

---

## 설치 및 실행 방법

이 프로젝트는 **Python**과 **Streamlit**으로 제작되었습니다. 아래 순서대로 따라 하시면 됩니다.

### 1. 필수 라이브러리 설치
터미널(CMD 또는 PowerShell)을 열고, 프로젝트 폴더로 이동한 뒤 아래 명령어를 입력하세요.
```bash
`python -m pip install -r requirements.txt`
```
### 2. 코드 및 서비스 실행
라이브러리 설치가 끝나면, 코드를 실행해준 뒤 디버그 콘솔에 아래 명령어를 직접 입력하고 엔터를 치세요.
```bash
streamlit run app.py
```
### 3. 접속하기
명령어를 입력하면 잠시 후 웹 브라우저가 자동으로 열리며 서비스가 시작됩니다.
